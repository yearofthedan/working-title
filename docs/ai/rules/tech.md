# Tech Standards

## Tech Stack

- **Framework:** Vue 3 (Composition API with `<script setup>`)
- **Language:** TypeScript
- **Build Tool:** Vite
- **Graph/Flow Engine:** [Vue Flow](https://vueflow.dev/) (`@vue-flow/core`, `minimap`, `controls`, `background`)
- **Rich Text Editor:** [Tiptap](https://tiptap.dev/) (`@tiptap/vue-3`, `starter-kit`)
- **Testing:**
  - Unit/Component: Vitest and Storybook
- **UI/Documentation:** Storybook
- **Linting/Formatting:** ESLint, Prettier
